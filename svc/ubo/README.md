# uBlock Origin list mirror

proxies all uBO lists via `https://services.helium.imput.net/ubo/lists/*`,
provides an assets.json
[here](https://services.helium.imput.net/ubo/assets.json)

### url

https://services.helium.imput.net/ubo/*

### privacy policy

this service does not log, track or store any personal information of any kind.

it keeps stats counters of: number of cache hits/misses, and cache size (number
of filter lists in cache, and how much memory they are taking up).

### maintenance

- `bump-version.sh` updates generation script to latest version
